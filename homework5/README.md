## 在本地端運行

若想在本地端執行本系統，請依照以下步驟：

```bash
conda create -n ecm python=3.11 -y
conda activate ecm
pip install -r requirements.txt

# 如果沒有 jupyter
conda install -c conda-forge jupyter -y
```

啟動 jupyter
```bash
conda activate ecm && jupyter notebook
```

執行 .py 檔案
```bash
conda activate ecm && python HW5.py
```
